# Capability Chain

## Core Chain

Concept
-> Task
-> Attempt
-> Evidence
-> Evaluation
-> Mastery
-> Standing

## Ownership

- Learner owns capability history
- Concept owns the knowledge domain
- Task owns the challenge
- Attempt owns learner performance on a Task
- Evidence owns observation
- Rubric owns interpretation criteria
- Evaluation owns judgment
- Mastery owns concept-level capability state
- Standing owns domain-level contextual capability position

## Key Principles

- Authority lives in the Evidence
- Evidence records what happened
- Rubrics explain what it means
- Evaluations preserver applied judgments
- Mastery accumulates Evaluations, not Evidence
- Standing contextualizes Mastery externally
- Observations are historical
- Interpretations are provisional
- Capability is multidimensional
