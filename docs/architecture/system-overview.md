# True Merit System Overview

## Core Entitites

### Concept
A Concept is the core unit of knowledge or skill that the learner is trying to understand.

### Task
A Task is an activity attached to a Concept that gives the learner an opportynity to demonstrate capability.

### Attempt
An Attempt is a learner's submitted response or performance on a Task.

### Evidence
Evidence is the observable proof extracted from an Attempt that shows what the learner can or cannot do.

### Rubric
A Rubric defines the criteria used to evaluate Evidence and determine the quality of the learner's performance.

### Mastery State
A Mastery State represents the learner's current level of demonstrated capability for a Concept.

### Standing
Standing is the broader learner status produced by combining mastery, evidence, and performance history across multiple Concepts.


## Relationship Between Entities

A Concept can have many Tasks.

A Task belongs to one Concept.

A learner can make many Attemps on a Task.

An Attempt produces Evidence.

Evidence is evaluated against a Rubric.

Rubric results update the learner's Mastery State.

Mastery States contribute to the learner's overall Standing.
