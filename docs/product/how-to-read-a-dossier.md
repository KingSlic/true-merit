# How To Read A True Merit Dossier

A True Merit dossier is a capability report.

It is designed to answer four questions:

1. What has this learner demonstrated?
2. What evidence supports that claim?
3. Where is the learner currently strong or weak?
4. What should the learner prove next?

Traditional systems usually show what someone completed.

True Merit attempts to show what someone can actually demonstrate.

---

## Evidence

Evidence is what was observed.

Evidence does not judge the learner.

Evidence records reality.

Examples:

- Learner implemented FastAPI CRUD endpoints.
- Learner explained the PATCH lifecycle from memory.
- Learner identified failure cases in a database relationship.
- Learner debugged OAuth session persistence.

Evidence answers:

> What happened?

---

## Evaluation

Evaluation is judgment applied to evidence.

An evaluation uses a rubric to decide what the evidence means.

Example:

Evidence:

> Learner explained the PATCH lifecycle from memory.

Evaluation:

> Reconstruction demonstrated.

Evaluation answers:

> What does this evidence show?

---

## Mastery

Mastery is the current interpretation of accumulated evaluations.

It does not come from one task.

It emerges over time.

Mastery answers:

> What can this learner reliably demonstrate now?

Common mastery dimensions include:

- Reconstruction
- Transfer
- Reasoning
- Independence
- Failure Awareness

---

## Standing

Standing is mastery viewed in context.

A learner may have different standings depending on the frame.

Examples:

- Backend Engineering Standing
- Bootcamp Cohort Standing
- Junior Developer Standing
- Job Posting Fit

Standing answers:

> What does this capability mean in this context?

---

## Recommended Next Proof

A dossier should not only describe the learner.

It should point toward the next proof of capability.

Examples:

- Add authentication.
- Write tests.
- Deploy the backend.
- Explain the architecture from memory.
- Transfer the same pattern to a new project.

Recommended Next Proof answers:

> What should this learner demonstrate next?

---

## Why This Matters

A True Merit dossier is not a resume.

A resume claims capability.

A dossier traces capability back to evidence.

The goal is not to make the learner look impressive.

The goal is to make capability visible, explainable, and improvable.
